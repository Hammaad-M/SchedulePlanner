Schedule Planner Tool 101

The Basics:

The Schedule Planner Tool is designed to break up series of tasks in equal sets of work seperated by adequate amounts of breaks to maximize productivity.

To-Do-List: The to-do-list is where you enter the jobs to complete with thier respective names and time required. The names are optional however the time required is not.

Time: This is where you enter the total time available to complete all tasks and have any breaks.

Result: This is where you will recieve an overview for the workflow after pressing "go".

Advanced Options:

Break between tasks: This is where you can decide if tasks may be halted to take a break or not.

Max Breaks: This allows you to set a limit for the maximum number of breaks.

Min Breaks: This allows you to set a limit for the minimum number of breaks.
